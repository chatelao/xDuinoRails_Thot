# Hardware

This directory contains the schematics, wiring diagrams, and other hardware-related files for the xDuinoRails_Thot signal decoder.

The hardware is designed to be flexible and adaptable to a variety of signaling needs. You will find:

*   **Schematics:** Detailed circuit diagrams for the decoder.
*   **PCB Layouts:** Gerber files and project files for the printed circuit board.
*   **Wiring Diagrams:** Examples of how to connect different types of signals to the decoder.

All hardware files are released under an open-source license, allowing you to build, modify, and share your own versions of the decoder.
